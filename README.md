# 🚗 Car Dekho Data Analysis using SQL

## 📌 Project Overview
This project focuses on analyzing a second-hand car dataset using SQL to derive meaningful business insights.

## 🎯 Objectives
- Count total number of cars
- Analyze year-wise car availability
- Find number of diesel cars in specific years
- Generate grouped reports using SQL

## 🛠️ Tools & Technologies
- SQL (MySQL / SQL Server)
- Data Analysis

## 📊 Key Insights
- Total cars available in dataset
- Year-wise distribution of cars
- Fuel-type analysis (Diesel cars count)
- Aggregated results using GROUP BY

## 🧠 SQL Concepts Used
- SELECT
- WHERE
- COUNT()
- GROUP BY
- ORDER BY
